# 배포 옵션 비교

## 현재 상황
Google Cloud SDK 설치 진행 중입니다. Cloud Run 배포를 위해서는 추가 설정이 필요합니다.

## 배포 옵션 비교

### 1. Google Cloud Run (현재 진행 중)
**장점:**
- 컨테이너 기반으로 확장성 우수
- 서울 리전 사용 가능
- 무료 티어: 월 200만 요청

**단점:**
- 초기 설정 복잡 (GCP 계정, 프로젝트, 결제 설정)
- gcloud CLI 설치 및 인증 필요
- Docker 이미지 빌드 및 푸시 과정 필요

**필요 사항:**
- Google Cloud 계정
- GCP 프로젝트 생성
- 결제 계정 연결 (무료 티어 사용 가능)
- gcloud CLI 설치 및 인증

---

### 2. Vercel (가장 간단 - 권장)
**장점:**
- GitHub 연동만으로 즉시 배포
- 자동 빌드 및 배포
- 무료 티어 충분 (월 100GB 대역폭)
- 커스텀 도메인 무료
- HTTPS 자동 적용
- 전 세계 CDN

**단점:**
- 서울 리전 없음 (하지만 CDN으로 빠름)

**배포 시간:** 5분 이내

**필요 사항:**
- GitHub 계정
- Vercel 계정 (GitHub로 로그인)

---

### 3. Netlify
**장점:**
- Drag & Drop으로 배포 가능
- GitHub 연동 가능
- 무료 티어 제공
- 자동 HTTPS

**단점:**
- Vercel보다 약간 느림

**배포 시간:** 5-10분

**필요 사항:**
- Netlify 계정

---

### 4. GitHub Pages
**장점:**
- 완전 무료
- GitHub 저장소만 있으면 됨
- 간단한 설정

**단점:**
- 정적 사이트만 가능 (현재 앱은 가능)
- 커스텀 도메인 설정 복잡

**배포 시간:** 10분

**필요 사항:**
- GitHub 계정 및 저장소

---

## 권장 사항

### 빠른 배포가 목적이라면: **Vercel** 추천
- 가장 간단하고 빠름
- 프로덕션 품질
- 무료로 충분

### 학습 목적이라면: **Cloud Run** 계속 진행
- 컨테이너 기술 학습
- GCP 경험
- 엔터프라이즈급 인프라

### 완전 무료가 중요하다면: **GitHub Pages**
- 결제 정보 불필요
- 영구 무료
